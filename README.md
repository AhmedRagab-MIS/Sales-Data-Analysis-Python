# Sales Data Analysis using Python

## 📊 Project Overview
This repository contains a comprehensive data analysis project focused on sales performance. The main objective is to clean, process, and visualize business data to uncover trends and extract actionable insights.

## 🛠️ Tech Stack & Libraries
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Tool:** Jupyter Notebook

## 📈 Key Analysis Features
- Data Cleaning and Handling Missing Values.
- Sales Performance Analysis across different regions and categories.
- Data Visualization including Sales Distribution (Pie Charts & Bar Plots).
